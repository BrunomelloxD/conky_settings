Informações úteis
=================

**SSD**
* **SSD / (raiz)** - `Espaço livre / Total espaço`  
* **SSD /home (home)** - `Espaço livre / Total espaço`

**MEMÓRIA RAM**
* **RAM** - `Memória em uso / Total de memória`

**MEMÓRIA VIRTUAL**
* **SWAP** - `Memória em uso / Total de memória`

**PROCESSADOR**
* **CPU** - `Uso total`
